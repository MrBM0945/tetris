# 🕹️ Tetris: Power-Up Edition

## 📝 Про проєкт

Цей проєкт — це реалізація класичної гри **Тетріс** мовою Python із використанням бібліотеки **PyGame**.

---

## 👥 Команда проєкту
* **Бабич Марія-Анастасія** - модуль фігур (Piece): форми, стани, операції переміщення і повороту
* **Бакаїм Максим**
* **Ірга Марія**
* **Миськів Богдан, поточні робота: логіка сітки тетрісу**

---

## 🚀 MVP (Мінімально життєздатний продукт)
Мінімальний набір функцій включає:
1.  **Візуалізація:** Прямокутне ігрове поле (сітка) з коректним відображенням кольорових блоків.
2.  **Динаміка:** Автоматичне падіння фігур з часом та їх зупинка при контакті з іншими блоками або дном поля.
3.  **Керування:** Повна підтримка клавіатури для переміщення фігур вліво/вправо та прискореного падіння.
4.  **Механіка обертання:** Можливість повертати фігурки навколо своєї осі для стратегічного планування.
5.  **Графіка:** Реалізація гри у віконному режимі (не консоль) з плавним оновленням кадрів.

---

* **Мова:** Python 3.x
* **Бібліотека:** PyGame
